Get QCA up and running in minutes.

## Prerequisites

- **Python 3.11+**
- **Node.js 18+** and **npm 9+**
- **NVIDIA API Key** from [NVIDIA API Catalog](https://build.nvidia.com/)

## Installation

### 1. Clone and Set Up Python Environment

```bash
git clone https://github.com/NVIDIA/ising-calibration-agent-cookbook.git qca
cd qca

# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# Install Python dependencies
pip install -e .
```

### 2. Install UI Dependencies

```bash
cd ui
npm install
cd ..
```

### 3. Configure Environment

Create a `.env` file in the project root:

```bash
cp .env.example .env
```

Edit `.env` and add your API key:

```bash
QCAL_SCRIPTS_DIR=./scripts
NVIDIA_API_KEY=nvapi-your-key-here
```

## Running QCA

QCA has three components: the backend server, the web UI, and the CLI.

### Option 1: Full System (Backend + Web UI)

**Terminal 1 - Start Backend:**
```bash
cd qca
qca serve
```

**Terminal 2 - Start Web UI:**
```bash
cd qca/ui
npm run dev
```

Open http://localhost:3000 in your browser.

### Option 2: CLI Only

For terminal-based interaction without the web UI:

```bash
source .venv/bin/activate
qca
```

### Option 3: Non-Interactive Commands

Run single commands without starting the TUI:

```bash
# List available experiments
qca experiments list

# Run an experiment
qca experiments run t1_measurement

# Check workflow status
qca workflow list
```

## Verify Setup

### Test Backend

```bash
curl http://localhost:8000/api/experiments/list
```

### Test CLI

```bash
qca experiments list
```

You should see a list of available experiments.

## Quick Commands Reference

| Command | Description |
|---------|-------------|
| `qca` | Launch interactive TUI |
| `qca -n "prompt"` | Run single command |
| `qca experiments list` | List experiments |
| `qca experiments run <name>` | Run experiment |
| `qca workflow list` | List workflows |
| `qca history list` | View experiment history |
