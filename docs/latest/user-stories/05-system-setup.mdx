Configuration, deployment, and administration.

## Stories

### SETUP-001 Install QCA

**As a** System Administrator,
**I want to** install QCA on a lab workstation,
**So that** researchers can use it for calibration.

**Acceptance Criteria:**
- [ ] Clear installation instructions in README
- [ ] Can install via pip or from source
- [ ] Dependencies are automatically resolved
- [ ] Installation can be verified with simple command

**Status:** Draft

---

### SETUP-002 Configure LLM Provider

**As a** System Administrator,
**I want to** configure which LLM provider to use,
**So that** the system uses our preferred AI service.

**Acceptance Criteria:**
- [ ] Can set API key via environment variable
- [ ] Supports NVIDIA, Anthropic, OpenAI, and local vLLM
- [ ] Can switch providers without code changes
- [ ] Clear documentation of provider options

**Status:** Draft

---

### SETUP-003 Set Custom Model

**As a** System Administrator,
**I want to** specify which model to use,
**So that** I can balance cost, speed, and quality.

**Acceptance Criteria:**
- [ ] Can set model via `QCA_MODEL` env var
- [ ] Default model works without configuration
- [ ] Invalid model gives clear error message

**Status:** Draft
