Conversational assistance and AI-guided calibration.

## Stories

### AGENT-001 Start Interactive Session

**As a** Quantum Researcher,
**I want to** start a conversational session with the agent,
**So that** I can work interactively on calibration tasks.

**Acceptance Criteria:**
- [ ] Can start TUI via `qca` command
- [ ] Can start web UI via `qca serve` + browser
- [ ] Agent greets user and indicates readiness
- [ ] Previous conversation context can be resumed

**Status:** Draft
